Music Discography Web App

This is a passion project I started in March 2020. It's a Spring web application and Rest API client that utilizes Apache HttpClient interact with Deezer.com's open-access API.

The search feature returns a list of Artists (Name, Picture, # of Deezer Fans) to choose from, matching the user's input string. Other features include: displaying discography of albums by the artist, and a album details including mp3 clips of tracks.
